Variables and Types

A variable is a location in the memory where you store values.
It has different types.

  1. int type
    1. int num = 10;
    2. int idNum = 00919;
  2. String type
    1. String name = “Fatema”;
    2. String lastName = “Bohra”;
  3. boolean type
    1. boolean is_female = true;
    2. boolean is_male = false;
